## Demo scene resource for [ROC engine](../../../roc_engine).

### Controls
* **W,A,S,D:** Movement.
* **Mouse wheel:** Change camera distance.
* **Tab**: Unlock cursor
* **Physics:**
  * **N:** Enable/disable physics.
  * **J:** Invert gravity.
  * **F1:** Physics debug.
  * **F2:** Change physics debug mode between 'normal' and 'X-ray'
